# NightWatch / 末夜守营 Demo

一个浏览器多人合作生存原型:

- 1 到 4 人房间制联机
- 白天采集木头、矿石、浆果
- 回营地按 `E` 强化营地
- 夜晚刷怪冲营
- 队友倒地后可以靠近按 `E` 救援
- 撑过 3 次夜袭获胜

## 启动

```bash
npm install
npm start
```

然后打开 [http://127.0.0.1:3000](http://127.0.0.1:3000)

## 部署到公网

这个项目已经补好了 [render.yaml](/Users/didi/Documents/NightWatch/render.yaml)，可以直接走 Render:

1. 把当前目录上传到 GitHub 仓库 `NightWatch`
2. 登录 Render，选择 `New +` -> `Blueprint`
3. 连接这个 GitHub 仓库
4. Render 会自动识别根目录里的 `render.yaml`
5. 部署完成后会给你一个公开网址，其他人直接打开就能联机进房

## 操作

- `WASD / 方向键`: 移动
- `鼠标`: 瞄准
- `空格 / 左键`: 攻击
- `E`: 存资源 / 救队友
